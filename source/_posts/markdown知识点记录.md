好记性不如烂笔头，MarkDown各种知识记录以备忘记。
<!--more-->

### 链接
```
1、内联方式
[picture](https://i.loli.net/2018/10/12/5bc034ce7b1de.jpeg)
2、引用方式
[picture][picture]

[picture]: https://i.loli.net/2018/10/12/5bc034ce7b1de.jpeg
```
效果：<br>1、内联方式
[picture](https://i.loli.net/2018/10/12/5bc034ce7b1de.jpeg)<br>2、引用方式
[picture][picture]<br>

[picture]: https://i.loli.net/2018/10/12/5bc034ce7b1de.jpeg

### 换行
```MarkDown
第一行<br>第二行
```
效果：<br>第一行<br>第二行

### 首行缩进
```MarkDown
[1] &nbsp;  半角的不断行的空白格
[2] &ensp;  半角的空格
[3] &emsp;  全角的空格
```
效果：<br>[1] &nbsp;  半角的不断行的空白格<br>[2] &ensp;  半角的空格<br>[3] &emsp;  全角的空格<br>
